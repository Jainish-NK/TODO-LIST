# 📝 ToDo List App

A simple ToDo List web app built using **HTML**, **CSS**, and **JavaScript**.

## ✨ Features
- Add, complete, and delete tasks
- Saves tasks using localStorage
- Responsive design

## 🚀 How to Use
1. Download or clone the repo
2. Open `index.html` in your browser

## 🛠️ Tech Used
- HTML5
- CSS3
- JavaScript
